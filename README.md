# ControleSg
> Controla o SgIntegra por web

## Dependencias
```sh
Python
 - psutil
 - flask
```

## Instalacao
Linux/Windows:
```sh
flask:
    - pip install flask  
psutil
    - pip install psutil
    - py -m pip install psutil
mysql-connect
    - pip install mysql-connector-python
paramiko
    - pip install paramiko
pythonping
    - pip install pythonping
```

## Uso:
    - Comando: python3 api.py
    - Interface web : http://127.0.0.1:5000
