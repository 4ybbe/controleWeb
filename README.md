```sh
Python
 - psutil
 - flask
 - mysql-connector-python

 - pythonping
```

## Instalacao
Linux/Windows:
```sh
flask:
    - pip install flask  
psutil
    - pip install psutil
    - py -m pip install psutil
mysql-connect
    - pip install mysql-connector-python
paramiko
    - pip install paramiko
pythonping
    - pip install pythonping
```

## Uso:
